Для подсветки части текста используется тег <mark> или класс .highlighted.

Варианты перечеркнутого текста:

  • удаленный текст <del>;
  • неточные данные <s>;
  • через класс ".strike".

Варианты подчеркнутого текста:

  1. дополнение к документу <ins>;
  2. подчеркнутая тегом <u>;
  3. через класс ".underline".

Отображение текста меньшего размера производится с помощью тега <small> или класса .small. Отображение текста большего размера — с помощью класса .large.

Жирный текст можно получить, используя тег <strong>, тег <b> или через класс .bold.
